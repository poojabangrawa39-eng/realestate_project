# 🏠 Real Estate Price Prediction System

## 📌 Project Overview

Real Estate Price Prediction System is a Machine Learning project that predicts the price of a property based on various features such as location, number of bedrooms, number of bathrooms, area size, and other property-related factors.

The project uses a trained Machine Learning model to analyze property details and provide an estimated house price through an interactive Streamlit web application.

## 🚀 Features

- Predict property prices instantly
- User-friendly Streamlit interface
- Machine Learning-based price estimation
- Real-time prediction results
- Easy property data input

## 📊 Dataset Features

The model uses property-related features such as:

- Number of Bedrooms
- Number of Bathrooms
- Area Size (Square Feet)
- Property Location
- House Age
- Number of Floors
- Nearby Facilities

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib

## ⚙️ How to Run

1. Clone the repository

```bash

```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```
